# User Management System With Admin Panel (Laravel Jetstream)

## Project Overview
This project is a **Laravel-based web application** using **Laravel Jetstream** for user registration, login, and role-based user management.  
It provides a modern, secure authentication system with an admin panel to manage users.

Key features include user authentication, user roles, profile management, and admin-controlled access.

---

## Features / Tasks

### User Features
- User Registration with email verification
- Login using email and password
- Profile management (update name, email, password)
- Secure session management
- Logout functionality
- Two-factor authentication (optional, via Jetstream)

### Admin Panel Features
- Admin login to access the dashboard
- View all registered users
- Edit user details
- Delete users
- Assign roles or permissions (optional)
- Monitor user activity

### Security Features
- Password hashing using Laravel's built-in bcrypt
- CSRF protection for forms
- Email verification for registration
- Role-based access control

---

## Installation / Setup

### Prerequisites
- PHP >= 8.x  
- Composer  
- MySQL / MariaDB  
- Node.js & npm (for frontend assets)
